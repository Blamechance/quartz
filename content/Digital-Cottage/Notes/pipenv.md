# Installation
```python
sudo apt-get remove python3-pipenv
sudo pip3 install pipenv (or sudo pip install pipenv)
```


# Commands
```sh
# To run/create pipfile
sudo pipenv install

# enter environment:
pipenv shell

# exit: 
deactivate


```
